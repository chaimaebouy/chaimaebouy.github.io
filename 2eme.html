Equations 2eme degré




c'est une équation de la forme ax² + bx + c = 0, où a, b et c sont des nombres connus avec a≠0 .Résoudre dans ℝ une équation d'inconnue x, c'est trouver les solutions réelles, c'est-à-dire les valeurs des réels x qui rendent l'égalité correcte.
Pour pouvoir résoudre une telle équation, il faut tout d'abord calculer le discriminant Δ.tels que Δ=b² - 4ac, puis il suffit de regarder le signe de Δ et de connaître le tableau suivant pour pouvoir conclure :



Δ = b²- 4ac
si Δ > 0
l'équation a deux solutions réelles
x1= (-b-√Δ)/(2a)
x2= (-b+√Δ)/(2a)		 si Δ = 0
l'équation a une solution unique réelle
x= -b/(2a)		 si Δ < 0
l'équation n'a aucune solution réell

Exemples :


1-

x² + x + 1 = 0 est une équation de degré 2; son discriminant est Δ = - 3; Δ est négatif et non nul.
Donc l'équation x² + x + 1 = 0 n'a pas de solution dans ℝ

2-

- x² + x + 30 = 0 est une équation de degré 2; son discriminant est Δ = 1² - 4(-30) = 121;
Δ est positif non nul, et Δ est le carré de 11.
Donc l'équation - x² + x + 30 = 0 admet 2 solutions dans ℝ x1=-5 et x2=6
